Intro to Objects

What is a JavaScript Object?

JS Objects are like objects IRL.

They have characteristics and things they can do.

CAR IRL

Characteristics

  • Make
  • Model
  • Year
  • Color

Things a Car Can Do:

  • Beep
  • Drive
Characteristics = Properties Things it can do = Methods

Object Literal Notation - Creating an Object in JS

        var myCar = {
            make: "Hyundai",
            model: "Kona",
            year: 2020,
            color: "Lime Twist"
        }
        // Get and set values in an object using dot notation.
        console.log(myCar.model);
        myCar.year = 2022;

        // Get and set values in object using bracket notation.
        console.log(myCar['model']);
        myCar["year"] = 2022;
