POSITION STICKY

The CSS Position Property

The first rule of the position property is DON'T ABUSE IT.

Very rarely will we ever need to use position.

All possible values of the position property are relative - it just becomes a question of what they are relative to.

position: static

Position static is the default for all HTML elements. Top, bottom, left, right have no effect.
static

position: relative

The element is positioned relative to its normal position. Top, bottom, left, right have effect.
relative

position: absolute

The element is positioned relative to its closest positioned ancestor. When you're absolutely positioning an element, you should also relatively position its parent.

position: fixed

The element is positioned relative to the scroll position.
